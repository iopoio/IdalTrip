행사2 + 관광지2 + 문화시설2 + 레포츠2 + 음식점2. 지역 데이터 불균형 문제 해결. 좋음.
메인에 "추천 코스 미리보기" 카드
킬러 피처. 축제 클릭 → 코스 미리보기 → 그대로 생성. 사용자 전환율 최고. 우선순위 1순위.
AI 추천 코스 기본 체크 + 버튼 텍스트 변경
"AI 추천 코스로 생성하기" 원클릭. 귀찮음 해소 앱 정체성에 딱 맞음.

⚠️ 데이터 품질 검증 필요. TourAPI 코스 데이터는 오래된 것 많음.
대안: 코스는 Gemini가 생성 (현재 방식 유지). TourAPI는 장소 데이터만.
