③ 실용적 데이터 흐름

1. 지역 선택 → areaBasedList2(25) → 코스 3~5개 목록
2. 코스 선택 → detailInfo2 → 서브장소 subcontentid 목록
3. 근처 장소 fetch 결과에서 contentid 교차 매칭
4. 매칭된 장소 → 한국관광공사 추천 뱃지 + 상단 체크
5. 미매칭 장소 → 하단 추가 선택 리스트
