# CLAUDE.md — 이달트립 (IdalTrip)

## 프로젝트 개요
- 시즌별 축제/체험 기반 여행 코스 자동 생성 서비스
- 공모전: 관광데이터 활용 공모전 14회 (관광공사+카카오)
- 마감: 2026.05.06 16시
- 컨셉: "이번 달 어디 갈까?" → 축제 추천 → 코스 자동 생성 → 카카오맵 연결

## 핵심 플로우
1. 메인: 월별/주별 축제 캘린더 → 추천
2. 축제 선택 → 해당 지역 관광지+맛집 목록 (AI 추천 + 사용자 체크)
3. 옵션: 내 위치 / 자차 or 대중교통 / 당일 or 1박2일 or 2박3일
4. AI가 일정 자동 생성 (자차/대중교통 2버전 동시)
5. 시간표 형태 일정표 + 각 장소 카카오맵 연결

## 기술 스택
- 프론트: React + TypeScript + Tailwind CSS
- 배포: Vercel (무료)
- 지도: 카카오맵 SDK + 카카오 경로 API (공모전 필수)
- 데이터: TourAPI 4.0 (축제 2만 + 관광지 5만 + 음식점 8만)
- AI: Gemini Flash API (코스 최적화/추천)
- 공공데이터: 공영주차장 API

## API 연동
| API | 용도 | 키 |
|-----|------|-----|
| TourAPI 4.0 | 축제/관광지/맛집/소요시간 | 공공데이터포털 발급 |
| 카카오맵 REST API | 경로 탐색(자차/대중교통) | 카카오 개발자 |
| 카카오맵 SDK | 지도 표시 + 내비 연결 | 카카오 개발자 |
| 공영주차장 API | 주차장 위치/요금 | 공공데이터포털 |
| Gemini Flash | 코스 추천/최적화 | Google AI Studio |

## 코스 생성 로직
1. 축제 선택 → 지역 관광지/맛집 30개 조회
2. Gemini가 대표 5~8개 관광지 + 2~3개 맛집 자동 추천
3. 사용자 체크/해제로 커스터마이징
4. 카카오 경로 API로 이동시간 계산
5. 영업시간/소요시간 고려 시간표 자동 배치
6. 자차: 공영주차장 연결 / 대중교통: 카카오 대중교통 경로
