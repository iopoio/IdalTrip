# 디자인 리뷰 — 스티치 v1 시안 검토

## 시안 구성 (4페이지)
1. 메인 홈 페이지
2. AI 로딩 및 추천 안내
3. 축제 상세 + 옵션 선택
4. 여행 코스 결과

---

## 살릴 것

- 오렌지(#FF6D00) + 블루(#0056D2) 조합 — 여행 서비스에 활기 있고 적절
- 벤토 그리드 축제 카드 레이아웃
- AI 로딩 → 완료 전환 구조
- 축제 상세의 체크박스 선택 UI (우리 플로우에 맞음)
- 타임라인 일정표 구조

---

## 수정 필요 사항

### 1. 컬러 토큰 불일치
4개 페이지마다 tailwind config 색상 정의가 전부 다름. 페이지 이동 시 색감이 달라지는 문제. 하나의 통일된 디자인 시스템 필요.

### 2. 한국어 서비스인데 영어 과다
"AI-Powered Season Discovery", "Best Route", "HOME/TRIPS/FESTS", "AI Curation Engine" 등. 타겟이 국내 20~30대인데 네비게이션까지 영어면 직관성 떨어짐. 브랜드명 "IdalTrip"만 영어, 나머지 전부 한국어.

### 3. "이달" 시즌감 부재
핵심 컨셉이 "이번 달"인데 메인 히어로가 generic 여행 앱. 지금이 몇 월이고 이번 달엔 뭐가 있는지가 첫 화면에서 안 읽힘. 월별 탭도 단순 pill 버튼 나열이라 캘린더/시즌 느낌 없음.

### 4. 타이포그래피 위계 붕괴
font-black, font-extrabold가 거의 모든 텍스트에 적용됨. 전부 굵으면 아무것도 강조가 안 됨. 제목(bold) > 섹션(semibold) > 본문(regular) > 캡션(light) 위계 필요.

### 5. 카카오맵 미포함
공모전 필수 요소인 카카오맵 경로 시각화가 코스 결과 페이지에 빠져 있음.

### 6. 모바일 하단 UI 충돌
BottomNav + FAB + 고정 액션바가 겹침. 축제 상세의 "AI 코스 생성하기" 버튼이 네비바를 완전히 덮음. 한 화면에 하단 고정 요소는 하나만.

### 7. 프로모션 섹션 로켓 아이콘
여행 서비스에 로켓은 맥락 안 맞음. 나침반/지도핀/경로 등 여행 메타포가 자연스러움.

### 8. 과잉 디자인
shadow-2xl 남발, rounded-[3rem] 과도한 라운딩. 카드 라운딩 최대 1.5rem, 그림자는 은은하게.
