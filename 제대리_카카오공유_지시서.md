## 배경

코스 완성 후 친구에게 여행 일정 공유 기능 추가.  
카카오 로그인 없이 JS SDK만으로 구현 가능.  
카카오맵 SDK와 카카오 공유 SDK는 별개 — 둘 다 로드 필요.

---

## Fix 1 — `src/main.tsx` 카카오 JS SDK 추가 로드

기존 `loadKakaoSDK` 함수 아래에 카카오 JS SDK 로드 추가:

```typescript
const loadKakaoShareSDK = () => {
  const script = document.createElement('script');
  script.src = 'https://t1.kakaocdn.net/kakao_js_sdk/2.7.2/kakao.min.js';
  script.crossOrigin = 'anonymous';
  script.onload = () => {
    if (!(window as any).Kakao.isInitialized()) {
      (window as any).Kakao.init(import.meta.env.VITE_KAKAO_JS_KEY);
      console.log('카카오 JS SDK 초기화 완료');
    }
  };
  document.head.appendChild(script);
};

loadKakaoShareSDK();
```

기존 `loadKakaoSDK()` 호출 아래에 `loadKakaoShareSDK();` 추가.

## Fix 2 — `src/pages/CourseResultPage.tsx` 공유 버튼 추가

### 타입 선언에 Kakao 추가

기존 `declare global` 블록에 추가:

```typescript
declare global {
  interface Window {
    kakao: any;
    Kakao: any;
  }
}
```

### 공유 함수 추가

`CourseResultPage` 컴포넌트 내부, `handleGenerateCourse` 위치 참고해서 추가:

```typescript
const handleShare = () => {
  if (!window.Kakao?.isInitialized()) {
    alert('카카오톡 공유 기능을 불러오는 중입니다. 잠시 후 다시 시도해 주세요.');
    return;
  }

  const placeList = currentSchedule
    .map((item, idx) => `${idx + 1}. ${item.place_name}`)
    .join('\n');

  window.Kakao.Share.sendDefault({
    objectType: 'feed',
    content: {
      title: course?.title || '이달의 여행 코스',
      description: `${course?.theme} · ${course?.total_duration}\n\n${placeList}`,
      imageUrl: 'https://images.unsplash.com/photo-1490750967868-88aa4486c946?q=80&w=800&auto=format&fit=crop',
      link: {
        mobileWebUrl: window.location.origin,
        webUrl: window.location.origin,
      },
    },
    buttons: [
      {
        title: '코스 보러가기',
        link: {
          mobileWebUrl: window.location.origin,
          webUrl: window.location.origin,
        },
      },
    ],
  });
};
```

### 공유 버튼 UI 추가

코스 제목 섹션(`<div className="px-5 pt-6 pb-4 bg-white">`) 내부, 제목 아래에 추가:

```tsx
<div className="flex items-center justify-between mt-3">
  <p className="text-sm font-semibold text-primary">{course.theme} · {course.total_duration}</p>
  <button
    onClick={handleShare}
    className="flex items-center gap-1.5 px-3 py-1.5 bg-[#FEE500] text-[#371d1e] rounded-xl text-xs font-extrabold active:scale-95 transition-transform"
  >
    <span>💬</span>
    카카오톡 공유
  </button>
</div>
```

기존 `<p className="text-sm font-semibold text-primary mt-1">` 줄은 제거하고 위 블록으로 교체.
