## 수정 1 — `src/pages/HomePage.tsx`

### 1-A. 히어로 섹션 교체

기존 대형 이미지 히어로 섹션(`<section className="relative w-full aspect-[4/5]...">`) 전체를 아래 텍스트 히어로로 교체:

```tsx
<section className="px-6 pt-8 pb-4 bg-white">
  <p className="text-xs font-bold text-primary tracking-widest uppercase mb-2">이달의 여행</p>
  <h1 className="font-headline text-3xl font-bold text-on-surface leading-tight whitespace-pre-line">
    {seasonCopy[currentMonth]?.title || `${currentMonth}월의 여행`}
  </h1>
  <p className="text-sm text-slate-500 mt-2 leading-relaxed">
    {seasonCopy[currentMonth]?.subtitle}
  </p>
</section>
```

```tsx

```

탐색 버튼 텍스트도 전체 선택 시 처리:
```tsx
{searchRegion === '전체' ? '전국' : searchRegion} · {날짜} 탐색하기
```

탐색하기 navigate 시 전체 처리 (ExploreResultPage에서 region=전체 처리는 기존 로직 확인 후 맞게):
```

```

### 1-C. 축제 섹션 지역 필터 탭 제거

`selectedRegion` 상태 및 `filteredFestivals` 필터 로직 전체 제거.  
`filteredFestivals` 대신 `festivals.slice(0, 8)` 바로 사용.  
축제 섹션 내 지역 탭 버튼 렌더링 블록 제거:

```tsx
// 이 블록 전체 제거
<div className="flex gap-6 mb-8 overflow-x-auto pb-2 scrollbar-hide">
  {regions.map(region => (...))}
</div>
```

`regions` 상수도 제거.

### 1-D. AI 맞춤 추천 섹션 전체 제거

```tsx
{/* AI Recommendation Section */}
<section className="pb-16 px-4">
  ...
</section>
```
이 섹션 통째로 제거.

### 1-E. 전체보기 버튼 레이아웃 수정

현재 `flex items-end justify-between` 블록에서 h2와 전체보기가 제대로 나란히 붙도록:

```tsx
<div className="flex items-center justify-between mb-6">
  <div>
    <h2 className="font-headline text-xl font-bold text-on-surface">현재 진행 중인 축제</h2>
    <p className="text-slate-500 text-xs mt-1">전국 각지의 생생한 축제 소식</p>
  </div>
  <button className="flex items-center gap-1 text-primary font-bold text-sm">
    전체보기
    <ArrowForward className="w-4 h-4" />
  </button>

```

지역 탭 블록은 이미 1-C에서 제거됨.

## 수정 2 — `src/components/MonthFilter.tsx`

현재 달 이전 월 제거. 현재 달부터 12월까지만 표시.  
내년 1월~은 이번 범위 밖.

```tsx
const today = new Date();
const thisMonth = today.getMonth() + 1;
const months = Array.from({ length: 12 - thisMonth + 1 }, (_, i) => thisMonth + i);
```

## 수정 3 — `src/components/BottomNav.tsx`

홈만 남기고 나머지 3개 제거:

```tsx
const navItems = [
  { name: '홈', path: '/', icon: Home },
];
```

탭 하나만 남으면 탭바 자체가 어색하므로 BottomNav 렌더링을 잠시 숨기는 것도 방법.  
판단: **탭바 컴포넌트는 유지하되 navItems를 비워서 렌더링 안 되게** 처리. App.tsx는 건드리지 말 것.

실제로는 아래처럼:
```tsx
const navItems: typeof allNavItems = []; // 임시 비움
```
또는 `return null;` 처리. 둘 중 빌드 통과하는 것으로.
