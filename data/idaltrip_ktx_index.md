# KTX 매거진 지식 인덱스 (정찰 보고서)

## 1. 사이트 개요
- **공식 홈페이지**: [ktxmagazine.kr](https://ktxmagazine.kr)
- **과월호 아카이브**: [ktxmagazine.kr/monthly](https://ktxmagazine.kr/monthly)
- **사이트 구조**: SSR (WordPress 기반). 모든 콘텐츠가 서버 사이드에서 렌더링되어 HTML로 전달됨.
- **콘텐츠 제공 방식**: 
  - 웹진: 개별 기사를 포스트 형태로 제공
  - PDF: 각 호별 전체 페이지 PDF 다운로드 링크 제공

## 2. 발행 정보
- **발행 빈도**: 월간 (Monthly)
- **발행일**: 매월 1일 (업로드는 전월 말경 진행됨)

## 3. 구조 및 페이지네이션
- **아카이브 방식**: `/monthly` 페이지에 연도별로 아카이브가 구성되어 있으며, 별도의 페이지네이션 없이 한 페이지에 최근 수년치(2022년~현재) 목록이 노출됨.
- **URL 패턴**: 
  - PDF: `https://ktxmagazine.kr/wp-content/uploads/{Year}/{UploadMonth}/{YYMM}.pdf`
  - 예시: 2026년 4월호 -> `https://ktxmagazine.kr/wp-content/uploads/2026/03/2604.pdf`
