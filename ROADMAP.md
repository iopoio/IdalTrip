## 제품 본질 (2026-04-15 후추님 정의)

**해결하는 문제**: "이번 주말에 어디 가고싶은데 어디가지?" — 축제 찾기 + 동선 + 숙소 + 맛집을 따로 검색하고 직접 짜야 하는 스트레스

**핵심 사용자**: 귀차니스트. 어디갈지 모르겠지만 주말에 나가고 싶은 사람.

**핵심 루프**:
```
발견 (축제 리스트 브라우징)
  → 탐색 (축제 상세 + 근처 맛집/관광지/숙박)
    → 계획 (AI가 일정 + 동선 자동 생성)
      → 이동 (카카오 네비 연결)
```

**데이터**: TourAPI 4.0 — 축제, 관광지, 음식점, 숙박, 여행코스, 레포츠, 쇼핑 전부 지원

**현재 채택된 방향 (공모전 마감 기준)**:
- 기존 루프(홈→축제상세→코스결과) 완성도 집중
- 카카오맵/네비 강조 (카카오 공동심사)
- 지도 핀 진입점(지역→지도→축제 선택)은 다음 버전
