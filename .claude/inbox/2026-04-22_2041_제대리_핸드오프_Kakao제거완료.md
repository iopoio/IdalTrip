## 작업 요약
Kakao 지도, 로컬, 모빌리티 API를 완전히 제거하고 Leaflet(OpenStreetMap) 및 한국관광공사 TourAPI(searchKeyword2 등)로 대체했습니다. 거리 계산은 Haversine 공식을 사용하도록 변경하여 공모전 마감(D-14) 전 403 Forbidden 문제를 해결했습니다.

## 변경 파일
- `vercel.json`: Kakao 관련 proxy(rewrite) 제거
- `.env`: Kakao API Key 주석 처리
- `src/types/index.ts`: Kakao 관련 타입(KakaoPlace, kakaoRating 등) 제거
- `src/services/kakaoMap.ts`, `src/utils/kakaoLoader.ts`: 전면 삭제
- `src/services/tourApi.ts`: `searchKeyword2` 엔드포인트 활용 추가
- `src/utils/distance.ts`: Haversine 및 이동 시간 예측 로직 신규 작성
- `src/pages/PlaceSelectionPage.tsx`: 맛집 검색 시 Kakao 대신 TourAPI `searchKeyword2`(`contentTypeId=39`) 활용하도록 변경
- `src/components/CourseMap.tsx`: `react-leaflet` 기반의 OSM(OpenStreetMap) 적용
- `src/pages/CourseResultPage.tsx`: "카카오내비 길찾기" 버튼 클릭 시 API가 아닌 KakaoMap 공개 URL(`map.kakao.com/link/map/`)로 연결하도록 수정
