## 🎯 1. Problem (문제)

### Top 3
1. **여행 일정 짜기 부담** — 가족·친구·연인 단위, 매번 수동
2. **축제·관광 정보 분산** — 네이버·다음·한국관광공사·블로그 검색 부담
3. **검색 결과 신뢰도 낮음** — 광고 위주 블로그, 가족 친화 필터 부재

### 기존 대안 (Existing Alternatives)
- 네이버·다음 검색 + 블로그
- 한국관광공사 사이트 직접 (코스 X)
- 트리플·마이리얼트립 (일반 관광 위주)
- ChatGPT 직접 (환각·UX 부담)
- Google Trips (서비스 종료)

## 💰 6. Revenue Streams (수익원)

### Phase 1 (1년)
- **광고** (AdSense 또는 자체 배너)
  - 가정: MAU 10만 × 광고 노출 5회/세션 × $0.001 CPC × 0.5% CTR = **월 $25** (보수적, 빠르게 확장)
- 무료 모델 유지 (진입 장벽 0)

### Phase 2 (2년차)
- **관광지 제휴** (특별 할인 → 수수료)
  - 가정: 월 1만 코스 × 5% 전환 × 평균 ₩30,000 × 5% 수수료 = **월 75만원**
- **프리미엄 구독** (광고 제거 + 가족 다중 프로필 + 오프라인 저장)
  - 가정: 월 ₩2,900, 1% 전환률 = MAU 10만 × 1% × ₩2,900 = **월 290만원**

### Phase 3 (3년차+)
- B2B (관광공사·지자체 라이선스)
- API 라이선스 (외부 앱 개발자)

## 📊 8. Key Metrics

### NSM
**월 자동 생성 + 실 사용된 코스 수**

### 5단계 깔때기 (AARRR)
| 단계 | 메트릭 | 1년 목표 |
|---|---|---|
| Acquisition | 월 방문자 | 30만 |
| Activation | 코스 생성 완료 | 10만 (33%) |
| Retention | 월 재방문 | 3만 (30%) |
| Revenue | 광고 클릭 또는 제휴 | $25/월+ |
| Referral | 코스 공유 | 5%+ |

## 🛡️ 9. Unfair Advantage (방어 자산)

1. **공공데이터 노하우** — 후추님 잡솔트·Publace에서 누적
2. **시즌드 사업자** — B2B·B2G 계약 가능 (1인 인디해커 차별)
3. **펩핀치 브랜드 신뢰** — 빌드인퍼블릭·공모전 검증
4. **한국관광공사 공식 API 활용도** — 가산점 영역 (공모전·기관 신뢰)
5. **한국 특화 가족 페르소나 디자인 톤** — 따뜻한 오렌지·코랄

### 약점 (외부)
- Google Trips 부활 시 직접 경쟁 (확률 낮음)
- 트리플·마이리얼트립이 축제 통합 추가 가능성
