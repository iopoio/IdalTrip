# 서비스 소개서 (사업 제안서) 골격 초안

> **목적:** 공모전 제출용 기획/서비스 소개 제안서
> **포맷:** PDF 변환용 문서 (PPT 또는 Notion 작성 권장)

## 1. Executive Summary (서비스 요약)
- **서비스명:** 이달여행 (IdalTrip)
- **한줄 소개:** AI 기반 시즌 축제 추천 및 맞춤형 여행 코스 자동 생성 서비스
- **타겟 유저:** 주말에 어디 갈지 고민하지만 계획 짜기는 귀찮은 '귀차니스트' 여행객
- **핵심 가치:** 발견(축제) → 탐색(장소) → 계획(AI 코스) → 이동(내비 연결)의 심리스한 경험 제공

## 2. Problem & Solution (문제 정의 및 해결책)
- **Problem:** 축제, 관광지, 맛집, 동선을 각각 다른 앱에서 검색하고 조합해야 하는 피로감.
- **Solution:** 
  - TourAPI를 활용해 매월/지역별 '축제' 중심의 큐레이션 제공.
  - Gemini AI를 활용해 영업시간, 소요시간, 거리를 고려한 '현실적인 일정표' 즉시 생성.

## 3. 핵심 기능 (Core Features)
- **지역/날짜 기반 축제 탐색:** TourAPI 4.0 축제 데이터 실시간 연동.
- **AI 동선 최적화 엔진:** 
  - 선택한 장소들을 바탕으로 Gemini가 최적의 시간표 설계.
  - 당일, 1박 2일 등 기간별 맞춤 코스.
- **직관적인 지도 UI & 스와이프 카드:** 
  - OpenStreetMap(Leaflet) 기반의 지도 마커 연동.
  - 카카오내비 다이렉트 연동으로 코스 완성 후 즉시 출발 가능.

## 4. 공모전 심사 기준 부합 요소 (평가 항목 맞춤 전략)
- **[데이터 활용성] TourAPI 4.0 핵심 매시업:** 축제(행사정보), 주변 관광지(키워드검색), 상세 개요(공통정보) 데이터를 AI 코스 엔진과 유기적으로 결합.
- **[창의성 및 혁신성] AI 큐레이션 + 오픈 생태계:** 단순 리스트업을 넘어 사용자 맞춤 타임라인 생성. 카카오 의존성을 벗어나 OSM(OpenStreetMap)으로 공개 API 생태계 재설계.
- **[논리성 및 완성도] 압도적인 UX와 즉시 응답성:** 카카오 라우팅 API 딜레이를 하버사인 기반 경량 자체 연산으로 대체하여 코스 생성 즉시 렌더링. Vercel 배포 기반 모바일 네이티브 UX 최적화.
- **[기대효과] 지역 경제 활성화:** 축제를 앵커(Anchor)로 주변 상권(맛집, 숙박)을 연결하여 체류형 관광 유도.

## 5. 비즈니스 파급력 및 향후 계획
- **기대 효과:** 지역 축제 활성화 및 체류형 관광 유도.
- **향후 로드맵:** 숙박 연동, 다국어 지원(인바운드 외국인 관광객 타겟), 공유하기 기능 강화.
