# 시연 영상 스크립트 초안

> **예상 분량:** 60 ~ 90초 내외
> **형식:** 모바일 화면 녹화 + 자막 (또는 나레이션)

---

### [00:00 - 00:10] 인트로 & 지역/날짜 선택
- **화면:** 메인 홈 화면. 부드럽게 스크롤하며 "이달의 축제로 떠나다" 헤드라인 보여줌.
- **액션:** 강원도 선택 → 4월 25일 날짜 선택 → '탐색 시작' 클릭.
- **자막:** "이번 주말, 어디 갈지 고민되시나요? 지역과 날짜만 선택하세요."

### [00:10 - 00:30] 장소 탐색 (TourAPI 연동)
- **화면:** 장소 선택 페이지 (PlaceSelectionPage). 
- **액션:** 
  1. 축제 리스트에서 1개 선택
  2. 주변 관광지 2개 체크
  3. 주변 맛집 1개 체크
- **자막:** "TourAPI가 추천하는 실시간 축제와 주변 관광지, 맛집을 원하는 대로 담기만 하세요."

### [00:30 - 00:45] AI 코스 생성
- **화면:** 'AI 추천 코스로 여행하기' 버튼 클릭 → 로딩 애니메이션.
- **자막:** "AI가 장소 간 거리와 영업시간을 분석해 완벽한 동선을 설계합니다."

### [00:45 - 01:10] 코스 결과 & 지도 연동 (OSM + 스와이프)
- **화면:** 코스 결과 페이지 로드.
- **액션:** 
  1. 상단 지도(OSM) 폴리라인 및 번호 마커 보여줌.
  2. 하단 카드를 옆으로 자연스럽게 스와이프. (스와이프 시 지도 마커 포커스 이동 확인)
- **자막:** "직관적인 지도와 타임라인 뷰로 하루 일정을 한눈에."

### [01:10 - 01:25] 카카오내비 출발 & 아웃트로
- **화면:** 카드 내 '카카오내비 길찾기' 노란색 버튼 클릭 → 카카오맵(웹/앱)으로 경로 열림.
- **자막:** "길찾기 버튼 하나로 목적지 안내까지 한 번에. 이달여행과 함께 지금 바로 출발하세요."
