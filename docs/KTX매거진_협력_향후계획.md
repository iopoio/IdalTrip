## 1. KTX 매거진 정체

- URL: https://ktxmagazine.kr/category/travel/
- 카드형 레이아웃, 15페이지 × 약 10글 = **누적 150글+**
- 발행일 표시 (월간 1~2회 추정)
- 글마다 지역·테마 명확 (해남·영월·군산·울주·일본 미식 등)
- 작성자 명시 (예: 고아라 글, 안홍범 사진)
- **제작 지원: 해당 지역 군청** (예: 해남편 = 해남군청 지원)

## 2. 콘텐츠 깊이 (해남편 분석 사례)

URL: https://ktxmagazine.kr/봄의-리듬을-따라-해남/

- **글 1편 = 3일 코스 1개 분량** (대흥사 → 윤선도 → 땅끝마을 → 우수영)
- 장소 25개+ (사찰·카페·숙박·박물관·자연)
- 사진 15장+
- 동선 명확 (시간 흐름)
- 메타데이터(주소·전화·운영시간) 부재 → IdalTrip이 보완 가능
