## 1. SWOT

### Strengths (강점)
- ✅ 한국관광공사 공식 API 활용 = 데이터 신뢰성
- ✅ 5초 자동 코스 생성 (네이버·다음 검색 30분~1시간 대비)
- ✅ 가족 친화 필터 (자녀 연령·접근성·소요 시간)
- ✅ PWA 모바일 우선 (앱 설치 부담 X)
- ✅ 시즌드 사업자 → B2B·B2G 계약 가능
- ✅ 따뜻한 오렌지·코랄 톤 (가족 페르소나 정합)

### Weaknesses (약점)
- 🔴 1인 운영 = 대규모 마케팅·CS 한계
- 🔴 사용자 코스 데이터 누적 X (초기) → 큐레이션 시간 필요
- 🔴 LLM API 의존 (Gemini 비용·정책 변경 위험)
- 🔴 코스 생성 정확도 [측정값 미확인] = 본선 전 검증 필수

### Opportunities (기회)
- 🟢 Google Trips 서비스 종료 → 한국 시장 공백
- 🟢 한국 축제 연 1,000+ → 통합 정보 수요
- 🟢 5/6 관광데이터 공모전 = 정부 신뢰·시드 자금 채널
- 🟢 외국인 한국 방문 회복 + K-Wave → 다국어 확장 시 글로벌 가능
- 🟢 모바일 우선 트렌드 (PWA 점유 확대)

### Threats (위협)
- 🟡 트리플·마이리얼트립이 축제 통합 추가 가능성
- 🟡 한국관광공사가 자체 코스 생성 서비스 출시 가능성 (정부 직접)
- 🟡 ChatGPT·Claude·Gemini 일반 챗봇이 직접 코스 생성 (UX 미흡 but 무료)
- 🟡 Google Trips 부활 (확률 낮음)
- 🔴 한국관광공사 API 정책 변경·유료화

## 2. PESTLE

| 축 | 영향 |
|---|---|
| **Political** | 관광 진흥 정책·지자체 축제 지원 → 🟢 우호. 단 공공 데이터 정책 변경 = 의존 위험 |
| **Economic** | 가계 여행 지출 회복 + 가성비 여행 트렌드 → 🟢 우호 |
| **Social** | 가족 시간 가치·짧은 주말 여행·자녀 연령별 활동 → 🟢 직격. 단 1인 가구 비중도 확대 (다른 페르소나 필요?) |
| **Technological** | LLM API 발전 + PWA 성숙 + 모바일 카메라·지도 API 안정 → 🟢 우호 |
| **Legal** | 개인정보보호법 (위치·일정) → 🟡 회원·로그인 도입 시 부담 |
| **Environmental** | 친환경 여행·로컬 관광 트렌드 → 🟢 차별화 가능 (지역 농산물 체험 등) |

→ **5축 우호 / 1축 부담** = 시장 진입 환경 좋음

## 3. Porter's Five Forces

### 1. 신규 진입 위협 — 🟡 중
- 진입 장벽: 한국관광공사 API 활용 노하우 + 코스 알고리즘 + 사용자 데이터 누적
- 단, LLM 일반 챗봇이 진입 시도 가능 (낮은 UX)
- 트리플·마이리얼트립 같은 기존 플레이어가 축제 통합 = 중간 위협

### 2. 구매자 교섭력 — 🔴 높음
- 사용자 = 무료 사용 (이탈 비용 0)
- 광고·구독 시도 시 즉시 이탈 가능
- → **첫 1년 무료 + 가치 검증 → 점진 BM** 필수

### 3. 공급자 교섭력 — 🟡 중
- 한국관광공사 API: 무료 공공데이터 = 🟢 낮음
- LLM (Gemini): 가격·정책 변경 위험 = 🟡 중. 단 다른 LLM (Claude·OpenAI) 갈아끼기 가능
- 지도 API (Kakao/Naver): 무료 티어 + 유료 = 🟡 중
- 도메인·호스팅 (Vercel): 무료 = 🟢 낮음

### 4. 대체재 위협 — 🔴 높음
- 네이버·다음 검색 + 블로그 (무료, 익숙)
- 트리플·마이리얼트립 (일반 관광)
- ChatGPT·Claude·Gemini 직접 (자유로움)
- 인스타·블로그 큐레이션
- → **차별점 = 한국관광공사 신뢰 + 가족 친화 + 5초 자동**으로 좁혀야

### 5. 경쟁 강도 — 🟡 중
- 직접 경쟁: 없음 (Google Trips 종료, 축제 통합 가족 친화 = 공백)
- 간접 경쟁: 트리플·마이리얼트립 (다른 페르소나)

## 4. Ansoff Matrix

| | 기존 시장 | 신규 시장 |
|---|---|---|
| **기존 제품** | **시장 침투** (Phase 1·2) — 한국 가족 단위 사용자 확장. SEO·SNS·공모전 본선 | **시장 개발** (Phase 3) — 외국인 한국 방문·해외 한국 거주자 (다국어 확장) |
| **신규 제품** | **제품 개발** (Phase 2~3) — 친구·연인·축제 마니아 페르소나·후기 평점·축제 캘린더 | **다각화** (Phase 4+) — 외국인용 K-여행 가이드·관광지 제휴 등 |

→ 전략 우선순위: **시장 침투 (1년) → 제품 개발 (2년차) → 시장 개발 (3년차) → 다각화 (4년+)**
