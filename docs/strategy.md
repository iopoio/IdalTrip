## 1. Vision (비전)
**한국 사용자가 축제·여행 일정 짜기 부담 없이 한 줄 입력으로 코스를 받는다.**
Google Trips(서비스 종료) 한국 공백 + 한국관광공사 공식 데이터 결합. 모바일 우선 PWA.

---

## 2. Segments (세그먼트)

| 우선순위 | 세그먼트 | 페인 | JTBD |
|---|---|---|---|
| 🔴 **P0 (코어)** | 가족 단위 여행자 (30~45세, 자녀 동반) | 일정 부담, 정보 분산 | "주말에 가족과 갈 만한 코스를 5분 안에 정하고 싶다" |
| 🟡 P1 | 친구·연인 그룹 (20~30대) | 큐레이션 부재 | "테마 여행 (축제·맛집·야경)을 자동 추천받고 싶다" |
| 🟢 P2 | 축제 마니아 (전국 추적자) | 통합 캘린더 부재 | "원하는 축제 조합 + 주변 여행지 같이 짜고 싶다" |

코어 P0 = "가족 단위" (시장 규모 + 결정권자 + 광고 ARPU 가장 높음).

---

## 3. Costs (비용 구조)

### 고정 비용 (월)
- Vercel Pro: $0 (무료 티어 충분)
- 도메인: 연 ~$15 (idaltrip.vercel.app 또는 별도 .com)
- Gemini API: ~$5~30/월 (사용량 기반)
- 한국관광공사 API: 무료 (공공데이터)
- 지도 API (Kakao/Naver): 무료 티어 (트래픽 모니터링)

### 변동 비용
- AI 코스 생성 = Gemini 토큰 사용량 비례
- 이미지·이미지 CDN = Vercel 무료 티어 한도 내

### 1인 운영 비용 = **월 ~$30 미만 가정** (효율충 노선 정합)

## 4. Value Propositions (가치 제안)

### 코어 가치
**"한 줄 입력 → 5초 안에 가족 친화 여행 코스"**

### 차별 가치 (다른 솔루션 X)
1. **한국관광공사 공식 데이터** = 신뢰성 (Google Maps 평점만으로 X)
2. **축제 캘린더 통합** = 분산 정보 통합 (네이버·다음 검색 부담 ↓)
3. **가족 친화 필터** = 자녀 연령·접근성 자동 반영
4. **PWA 모바일 우선** = 앱 설치 부담 X

### 가치 제안 (vs 대안)
| 대안 | 한계 | IdalTrip 차별 |
|---|---|---|
| 네이버·다음 검색 | 일정 짜기 수동 | 자동 코스 생성 |
| 한국관광공사 사이트 | 데이터만 노출, 코스 X | 코스 자동 + 일정 표 |
| 트리플·마이리얼트립 | 일반 관광 위주, 축제 통합 X | 축제 강조 + 가족 |
| Google Trips | 서비스 종료 | 한국 공백 점유 |
| ChatGPT 직접 | API 비용·환각·UX 부담 | 검증된 데이터 + UI |

---

## 5. Trade-offs (트레이드오프)

| 트레이드오프 | 결정 | 이유 |
|---|---|---|
| 정확도 vs 속도 | **속도 우선 (5초 안에 응답)** | 사용자 이탈률 ↓ |
| 자동화 vs 수정 | **자동 + 수정 가능** | 자동 코스 + 사용자 항목 추가·삭제 |
| 광고 vs 구독 | **광고 우선 (1년)** | 무료 진입 장벽 0, MAU 우선 |
| PWA vs 네이티브 앱 | **PWA 우선** | 설치 부담 X, 즉시 사용 |
| 한국 vs 글로벌 | **한국 우선** | 한국관광공사 차별점, 글로벌은 후속 |
| 깊이 vs 넓이 | **깊이 (가족 P0 코어)** | 광고 ARPU·결정권자 우선 |

## 6. Metrics (지표)

### North Star Metric (NSM)
**"월 자동 생성 + 실 사용된 코스 수"** (단순 생성 X, 실 방문·공유까지)

### Input Metrics
| 메트릭 | 1년 목표 |
|---|---|
| MAU (월 활성 사용자) | 10만 |
| 코스 생성 성공률 | 99%+ |
| 평균 응답 시간 | 3초 이내 |
| 만족도 (NPS) | 40+ |
| PWA 설치율 | 10%+ |
| 재방문률 (월간) | 30%+ |

### Health Metrics (위험 신호)
- API 에러율 ≥1% → 알림
- Gemini 토큰 사용량 월 $50 초과 → 모델 다운그레이드 검토
- Vercel 무료 한도 80% 도달 → 유료 검토

## 7. Growth (성장)

### 단기 (3개월)
- SEO: "축제 여행 코스" "주말 가족 여행" 키워드
- X·인스타: 사용자 코스 공유 → 자연 바이럴
- 펩핀치 사이트 = 진입로

### 중기 (6개월)
- 본선 진출 시 보도자료
- 관광공사·지자체 협업 (공모전 본선 후)

### 장기 (1년+)
- 모바일 앱
- 다국어 (영어·중국어·일본어 — 외국인 한국 방문)
- 관광지 제휴 (특별 할인)

## 9. Defensibility (방어 자산)

### 장기 방어
1. **공공데이터 활용 노하우** = 다른 솔루션 진입 시 학습 비용
2. **사용자 코스 데이터** = 시간 누적 → 큐레이션 우위
3. **펩핀치 브랜드 신뢰** = 공모전 본선·라이브 검증
4. **시즌드 사업자** = B2B·B2G 계약 가능
5. **한국 특화 디자인** = 따뜻한 톤·가족 페르소나

### 약한 방어
- LLM API 의존 (Gemini → 다른 모델 갈아끼기 가능)
- 한국관광공사 API 정책 변경 위험 (공공이라 안정적)
- 코스 알고리즘 = 모방 가능 (단 사용자 데이터·UI는 카피 어려움)
