## 1. Executive Summary

IdalTrip은 한국 축제 정보와 관광 데이터를 기반으로 **여행 코스를 자동 생성**하는 웹 서비스다. Google Trips(서비스 종료) 한국 공백을 메우고, 한국관광공사 공식 API로 데이터 신뢰성을 확보한다. 본 사업은 광고·관광지 제휴 수익 + 향후 모바일 앱 확장이다.

### 공모전

## 3. Objectives & Success Metrics

### Goals (본 사업)
1. 한국 사용자가 축제·여행 코스를 5초 안에 받음
2. 자동 생성 코스 만족도 70%+
3. 월 활성 사용자 (MAU) 10만 (1년 내)
4. 광고·관광지 제휴 수익 모델 검증

### Goals (현 Phase 5/6 공모전)
1. 5/6 출품 가능 수준 안정화
2. 라이브 데모 시연 시 코스 생성 성공률 ≥95%
3. 공식 API 활용도 명시 (가산점 영역)
4. 본선 진출

### Non-Goals
1. 자체 숙박·예약 (정보 + 외부 링크만)
2. 호텔·항공 통합 (별도 영역)
3. 글로벌 출시 (한국 우선)
4. 사용자 SNS·리뷰 (별도 사업, MVP 제외)
5. 모바일 네이티브 앱 (PWA·웹 우선)

## 4. Target Users & Segments

### Segment 1 — 가족 단위 여행자
- 페르소나: 30~45세, 자녀 동반 주말 여행
- 페인: 일정 짜기 부담, 축제 정보 분산
- 가치: 자동 코스 + 가족 친화 필터

### Segment 2 — 친구 그룹 여행자
- 페르소나: 20~30대, 주말·연휴
- 페인: 연인·친구 대상 코스 큐레이션 부재
- 가치: 자동 추천 + 공유 기능

### Segment 3 — 축제 마니아
- 페르소나: 특정 축제 추적, 연 5+ 회 방문
- 페인: 축제 일정 분산, 통합 캘린더 부재
- 가치: 축제 캘린더 + 주변 여행지 조합

### Segment 4 — 공모전 심사위원 (즉시 타겟)
- 평가 포인트: 한국관광공사 API 활용도, 사회적 가치, 기술 완성도
- 가치: 라이브 시연 + API 도식

## 6. Solution Overview

### 핵심 기술
- React + Vite + Tailwind
- Vercel 배포 (idaltrip.vercel.app)
- 한국관광공사 API (TourAPI)
- AI 코스 생성: Gemini API [모델 ID 코드 직접 확인]
- 지도: Kakao Maps 또는 Naver Maps [코드 직접 확인]

### 데이터 흐름
1. 사용자 입력 (출발지·날짜·인원)
2. TourAPI로 축제·관광지 조회
3. AI가 코스 생성 (이동 시간·테마 매칭)
4. 지도 + 일별 표 시각화
5. 외부 링크 (지도·맛집·숙박)

### API 활용 (공모전 가산점)
- TourAPI 축제 정보, 지역 관광지, 카테고리별 데이터
- JSON 우선, 페이징 처리, 에러 코드 (99=서비스, 12=URL)

### 디자인 방향
- 따뜻한 오렌지·코랄 (디자인-가이드.md 정합)
- 지도 강조 (Kakao/Naver 임베드)
- 카드 기반 일정 + 카드 = `rounded-xl + shadow-md + bg-white`
- 모바일 퍼스트, 터치 타겟 44×44px

## 8. Timeline & Phasing

### Phase 0 — 5/6 공모전 출품 (즉시 ~ 2026-05-06)
- Phase 5 잔여 작업 마무리
- 코스 생성 안정화 (성공률 95%+)
- 라이브 데모 시연 시나리오
- 사업계획서 (Cowork 매니저 영역)

### Phase 1 — 본선 진출 후 (5월 ~ 7월)
- 사용자 피드백 수렴
- P1 기능 추가 (공유·캘린더·저장)
- API 응답 캐싱·성능 최적화

### Phase 2 — 본 사업 진입 (8월 ~ 연말)
- 광고 SDK 시드 (AdSense)
- 관광지 제휴 1~2건 시범
- 사용자 코호트 분석

### Phase 3 — 사업 확장 (2027~)
- PWA 설치 + 모바일 앱
- 다국어
- 후기·평점 + 회원 시스템

### 후추님 1인 노선 정합성
- 1~2개 집중 메모리 = 본선 결과 보고 다른 프로젝트와 우선 비교
- 공모전 = 시드 자금·신뢰 자본 = 본 사업 진입 가속화
